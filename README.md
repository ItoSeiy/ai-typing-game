# MATRIX TYPER

Matrix風タイピングゲーム（寿司打ライク）

## ゲーム概要

**MATRIX TYPER** は、Matrix 風の視覚エフェクトと日本語ローマ字入力を組み合わせたブラウザゲームです。

- 画面上方から降り注ぐ文字列を、表示されるローマ字に沿って高速に入力
- 正答率と入力速度でスコア加算
- `START` から開始し、ゲーム終了時に成績を表示
- `src` 未改変で、公開時でもゲーム動作を損なわないモジュラー構成

## 遊び方

1. ブラウザで `index.html` を開く
2. `START` ボタンで開始
3. 画面に現れるローマ字をタイプ（英字入力）
4. 入力完了・ミス時の判定に応じて加点・減点
5. ゲームオーバー後、結果画面でスコアを確認

## スクリーンショット風説明（テキスト）

- 背景: 暗い夜空を思わせるダークテーマに、緑のマトリクス文字列が流下
- 中央上部: タイトルロゴ「MATRIX TYPER」＋ START/RESET ボタン
- 中央: レーン上に流れる対象語（ひらがなとローマ字ヒント）
- 左: コンボとレベル表示
- 右: 残機・速度表示
- 下部: 入力欄、正答率、ベストスコア、ラウンド進行インジケータ
- 右上: 一時停止/リトライ導線

## 特徴

- Matrix 風ビジュアル演出（Canvas ベース）
- Web Audio API による効果音再生（正解・ミス・ゲーム開始など）
- CSV 駆動のレベルデザイン（データ変更で難易度を調整）
- モジュラー構成で、UI/音声/データ/ゲーム進行の責務分離
